Das ist fast alles. Wenn die aktuelle n immer noch größer als 1 ist, geben Sie ein Produkt von n zurück und rufen Sie die gleiche Funktion mit dem Argument n-1 auf (rekursiver AnsatzL mehrfacher verschachtelter Aufruf der gleichen Funktion). 

return n*factorial(n-1) if n> 1 else 1
Wie sollte also meine endgültige Lösung aussehen?

Hier ist die gesamte Lösung:

def factorial(n: int) -> int:

    return n*factorial(n-1) if n> 1 else 1

Beginnen Sie bei 1 und multiplizieren Sie mit allen ganzen Zahlen von 2 bis zu der angegebenen n. 

def factorial(n: int) -> int:

    res = 1
    for i in range(2, n + 1):
        res *= i
        
    return res
