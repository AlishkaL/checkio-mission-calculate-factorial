差不多就是这样。如果当前n 仍大于 1，则返回n 的乘积，并调用带有n-1 参数的同一函数（递归方法L 多次嵌套调用同一函数）。 

return n> 1? n * factorial(n - 1): 1;
那么，我的最终解决方案应该是怎样的呢？

你只需将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

function factorial(n: number): number {

    return n> 1? n * factorial(n - 1): 1;
}

从 1 开始，乘以从 2 到给定n 的所有整数。 

function factorial(n: number): number {

    let res = 1;
    for (let i = 2; i <= n; i++) 
        res*= i;
    
    return res;
}
