Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę użyć podejścia rekurencyjnego: gdy n jest mniejsze niż 2, funkcja musi zwrócić 1, w przeciwnym razie zwraca iloczyn n i czynnika kolejnej mniejszej liczby całkowitej.

Ok, od czego powinienem zacząć?

Proszę napisać ternary operator z tymi dwoma przypadkami. 

return n> 1? ...: ...;
Co robić dalej?

W przypadku n 0 lub 1 funkcja musi zwrócić 1. 

return n> 1? ...: 1;
Nadal potrzebuję podpowiedzi.

To prawie wszystko. Jeśli bieżący n jest nadal większy niż 1, proszę zwrócić iloczyn n i wywołać tę samą funkcję z argumentem n-1 (podejście rekurencyjneL wielokrotne zagnieżdżone wywołanie tej samej funkcji). 

return n> 1? n * factorial(n - 1): 1;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę tylko opakować swój kod w wyrażenie funkcyjne. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

function factorial(n: number): number {

    return n> 1? n * factorial(n - 1): 1;
}

Jasne, proszę pobrać!

Proszę zacząć od 1 i pomnożyć przez wszystkie liczby całkowite od 2 do podanej n. 

function factorial(n: number): number {

    let res = 1;
    for (let i = 2; i <= n; i++) 
        res*= i;
    
    return res;
}
