Was ist als nächstes zu tun?

Für n 0 oder 1 muss Ihre Funktion 1 zurückgeben. 

return n> 1? ...: 1;
Ich brauche noch einen Hinweis.

Das ist fast alles. Wenn die aktuelle n immer noch größer als 1 ist, geben Sie ein Produkt von n zurück und rufen dieselbe Funktion mit dem Argument n-1 auf (rekursiver AnsatzL mehrere verschachtelte Aufrufe derselben Funktion). 

return n> 1? n * factorial(n - 1): 1;
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen Ihren Code nur in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function factorial(n: number): number {

    return n> 1? n * factorial(n - 1): 1;
}

Klar, schnappen Sie es sich!

Beginnen Sie bei 1 und multiplizieren Sie mit allen ganzen Zahlen von 2 bis zu der angegebenen n. 

function factorial(n: number): number {

    let res = 1;
    for (let i = 2; i <= n; i++) 
        res*= i;
    
    return res;
}
